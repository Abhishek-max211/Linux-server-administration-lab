# 📜 Logs Monitoring

> **Module 12** of the **Linux Administration Lab**

## 📖 Overview

Logs are one of the most important resources for Linux System Administrators. They help monitor system activities, troubleshoot issues, investigate authentication attempts, analyze service behavior, and diagnose system failures.

In this lab, I explored the Linux log directory, identified important log files, monitored system logs using `journalctl`, viewed service-specific logs for Nginx, and inspected recent system events.

---

## 🎯 Objectives

In this lab, I performed the following tasks:

- Explore Linux log files
- Identify important system logs
- Search log files using `grep`
- View systemd journal logs
- Monitor service-specific logs
- Inspect recent system events

---

## 💼 Real-World Scenario

You are working as a **Linux System Administrator** at **TechNova Pvt. Ltd.**

Users report intermittent issues with the company web server. Your responsibility is to investigate system logs, review Nginx service logs, and inspect recent journal entries to identify possible errors or service-related events.

## Task 2 – Search Important Log Files

Located logs related to Ubuntu.

```bash
ls | grep "ubuntu"
```

Located Nginx logs.

```bash
ls | grep "nginx"
```

Located authentication logs.

```bash
ls | grep "auth"
```

Located boot logs.

```bash
ls | grep "boot"
```

Located system activity logs.

```bash
ls | grep "sys"
```

---

## Task 3 – View System Journal

Displayed complete system journal.

```bash
journalctl
```

---

## Task 4 – View Nginx Service Logs

Displayed logs generated by the Nginx service.

```bash
journalctl -u nginx
```

---

## Task 5 – View Recent System Events

Displayed detailed recent system events and errors.

```bash
journalctl -xe
```

# 🛠 Commands Explained

| Command | Purpose |
|----------|----------|
| `cd /var/log` | Navigate to Linux log directory |
| `ls` | List available log files |
| `grep` | Search for specific log files |
| `journalctl` | View complete systemd journal |
| `journalctl -u nginx` | View logs for the Nginx service |
| `journalctl -xe` | Display recent system events with detailed information |

---

# 📂 Important Linux Log Files

| Log File | Description |
|----------|-------------|
| `auth.log` | Authentication and login activity |
| `syslog` | General system messages |
| `kern.log` | Linux kernel messages |
| `bootstrap.log` | System boot process |
| `dmesg` | Kernel ring buffer messages |
| `journal/` | Binary logs managed by systemd |
| `nginx/` | Nginx web server logs |
